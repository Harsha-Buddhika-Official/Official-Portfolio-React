# Official Portfolio - Harsha Buddhika

A modern, responsive personal portfolio website showcasing the work and skills of Harsha Buddhika, a Computer Science undergraduate specializing in Full-Stack Development, Cybersecurity, and Game Development. Built with React.js and Tailwind CSS for optimal performance and visual appeal.

## ✨ Features

- **🎨 Modern Design**: Clean, professional interface with smooth animations and hover effects
- **📱 Fully Responsive**: Optimized for all devices - desktop, tablet, and mobile
- **⚡ Fast Performance**: Built with Vite for lightning-fast development and production builds
- **🔄 Dynamic Typing Animation**: Eye-catching typewriter effect showcasing multiple roles
- **🎯 Smooth Navigation**: Seamless scrolling between sections with fixed navigation bar
- **🌙 Dark Mode Ready**: Built with dark mode compatibility using Tailwind CSS
- **📊 Interactive Skill Cards**: Animated skill showcase with technology icons
- **💼 Project Portfolio**: Detailed project cards with GitHub integration
- **📧 Contact Integration**: Direct contact form and social media links

## 🗂️ Sections

- **🏠 Home**: Dynamic introduction with animated typing effect showcasing roles:
  - Computer Science Undergraduate
  - Software Engineer
  - UI/UX Designer
  - Full Stack Developer
  - Web Developer
  - Graphic Designer

- **👨‍💻 About**: Personal introduction highlighting expertise in Full-Stack Development, Cybersecurity, and Game Development

- **💼 Projects**: Showcase of development projects including:
  - **ToDoApp - React**: Interactive task management application with real-time updates
  - **Modern Portfolio Site**: This very portfolio showcasing responsive design and modern architecture

- **🛠️ Skills**: Interactive skill cards featuring technologies and tools with smooth animations

- **🎓 Education**: Academic background and certifications

- **📞 Contact**: Multiple ways to get in touch with direct contact form integration

## 🚀 Quick Start

### Prerequisites

- **Node.js** (v16 or higher)
- **npm** or **yarn** package manager
- **Git** for version control

### Installation Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Harsha-Buddhika-Official/Official-Portfolio-React.git
   ```

2. **Navigate to project directory**:
   ```bash
   cd Official-Portfolio-React
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start development server**:
   ```bash
   npm run dev
   ```

5. **Open your browser** and visit `http://localhost:5173`

## 🛠️ Available Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build production-ready application
- `npm run preview` - Preview production build locally
